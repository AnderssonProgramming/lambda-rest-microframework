🚀 Lambda REST Microframework

A lightweight Java web framework with lambda-based REST services

✨ Features

📡 REST Services

Define REST endpoints with lambda functions using get()

🔍 Query Parameters

Extract query values with req.getValues()

📁 Static Files

Serve HTML, CSS, JS, and images from configurable directories

⚡ Lightweight

No external dependencies — pure Java with raw sockets

🧪 Try the REST API

Click the button to test the API...

Available Endpoints

Endpoint Description Action
/App/hello?name=Pedro Greeting service with query parameter
/App/pi Returns the value of PI
/App/time Returns current server time
/App/echo?msg=test Echoes query parameters
